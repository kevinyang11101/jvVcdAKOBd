# 食堂订餐小程序SSM

## 前言

随着互联网技术的不断发展，食堂订餐也逐渐步入了数字化时代。本项目的食堂订餐小程序基于SSM框架开发，结合微信小程序，为广大用户提供便捷、高效的订餐体验。

## 内容介绍

本项目是一款适用于校园或企业内部的食堂订餐小程序，主要功能包括：用户注册登录、菜品浏览、购物车管理、订单提交与支付等。通过本项目，用户可以轻松实现线上订餐，节省时间成本，同时提高食堂运营效率。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
